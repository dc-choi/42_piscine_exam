Expected files   : sort_int_tab.c
Allowed functions: 
--------------------------------------------------------------------------------

Write the following function:

void sort_int_tab(int *tab, unsigned int size);

정확하게 'size' 멤버를 포함하는 'tab' int 배열은 오름차순으로 정렬(in-place)해야 합니다.

복식은 보존해야 합니다.

입력은 항상 일관성이 있습니다.
