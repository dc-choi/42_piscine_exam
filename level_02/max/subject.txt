Assignment name  : max
Expected files   : max.c
Allowed functions: 
--------------------------------------------------------------------------------

Write the following function:

int		max(int* tab, unsigned int len);

첫 번째 매개 변수는 int의 배열이고 두 번째는 배열의 요소 수입니다.

함수는 배열에 있는 가장 큰 수를 반환합니다.

배열이 비어 있으면 함수는 0을 반환합니다.
